Answer the questions:
1.Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

- предварительно должен быть установлен GIT на локальный компьютер.
- зайти на https://github.com/, выбрать нужный репозиторий, нажать кнопку "Code", скопировать ссылку
- открыть Git Bush, выбрать папку где разместить репозиторий, ввести команду git clone https://ссылка на репозиторий.git

2.Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

- выбрать или создать каталог для репозитория
- открыть Git Bush, открыть этот каталог
- git init
- git add
- git commit -m "initial commit"
- зайти на сайт github, создать репозиторий, скопировать ссылку на него
- git remote add origin <ссылка на репозиторий>
- git push -u origin main(или master) 

3.Как получить на локальный компьютер изменения с github.com

- открыть Git Bush, открыть этот каталог c репозиторием
- git pull

4.Какую информацию нам дают команда :
git status - показывает, какие файлы были изменены, добавлены или удалены, но еще не добавлены в индекс, а также какие файлы были добавлены в индекс и готовы для коммита.
git log - показывает историю коммитов
git remote -v - показывает список удаленных репозиториев, связанных с текущим локальным репозиторием.
